# Welcome to Hubbly

Hubbly is a comprehensive lobby management plugin for Minecraft servers running Spigot/Paper 1.20+. It provides everything you need to create an engaging and polished lobby experience for your players.

## Key Features

<CardGroup cols={2}>
  <Card title="Movement Enhancement" icon="rocket">
    Advanced movement mechanics including double jump, launchpads, tridents, enderbows, grappling hooks, and the Aspect of the End teleportation.
  </Card>
  
  <Card title="Player Management" icon="users">
    Comprehensive player controls with visibility toggling, forced inventory management, custom join/leave messages, and anti-void protection.
  </Card>
  
  <Card title="Custom Items & Menus" icon="box">
    Create custom items with actions, interactive menus, server selectors, and configurable hotbar layouts that execute on player join.
  </Card>
  
  <Card title="Chat Management" icon="comments">
    Full chat control with chat locking, clearing, word filtering, command blocking, and automated announcements with RGB gradient support.
  </Card>
  
  <Card title="World Protection" icon="shield">
    Complete world event management to prevent block breaking/placing, item drops, mob spawning, damage, and more with per-world configuration.
  </Card>
  
  <Card title="Database Integration" icon="database">
    Optional MySQL/MariaDB support with HikariCP connection pooling for persistent data storage across server restarts.
  </Card>
</CardGroup>

## What Makes Hubbly Special?

<Note>
Hubbly is designed with performance and customization in mind. Every feature can be toggled, configured, and fine-tuned to match your server's unique needs.
</Note>

- **RGB Color Support**: Full hex color and gradient support throughout the plugin
- **PlaceholderAPI Integration**: Use placeholders in messages, menus, and announcements
- **Action System**: Powerful action framework for items and events (titles, sounds, fireworks, commands, menus)
- **BossBar Animations**: Animated boss bars with customizable colors and text rotation
- **Anti-WDL**: Built-in WorldDownloader protection to secure your lobby
- **Update Checker**: Automatic update notifications for administrators
